# JerryScript Debugger Tests
Auto tests for Zephyr.js JerryScript debugger

## Dependencies

- node.js
- `yarn` or `npm` (the steps below use `yarn`)

## Installing

```
$ cd jrs-debugger-tests
$ yarn install
```

## Pre-condition


## Running tests

```
$ cd jrs-debugger-tests
$ yarn test
```

## Running tests in watch mode
```
$ cd jrs-debugger-tests
$ yarn test:watch
```

## Running the linter

```
$ cd jrs-debugger-tests
$ yarn lint
```
